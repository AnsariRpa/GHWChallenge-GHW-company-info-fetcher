# GHWChallenge-GHW-company-info-fetcher
GHW API Challenge - Company Info Fetcher

A simple API-based web app that fetches company logos and basic information using a domain name.

Built as part of Global Hack Week (GHW) API Challenge.

---

## Features

- Enter a company domain (e.g. google.com)
- Fetch company logo dynamically
- Uses external APIs for real-time data
- Fallback mechanism for reliability
- Clean and simple UI

---

## Tech Stack

- Python
- Streamlit
- REST APIs
- GitHub Copilot

---

## GitHub Copilot Usage

GitHub Copilot was used to:
- Generate helper functions (e.g. domain validation)
- Assist with API integration logic
- Speed up development by suggesting code snippets

---

Notes
Uses Clearbit Logo API and Google Favicon API as fallback
Handles invalid inputs gracefully
